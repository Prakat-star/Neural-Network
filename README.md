# Neural Network for MNIST Digit Recognition
# Overview
This project implements a simple neural network for recognizing handwritten digits using the MNIST dataset. It also includes an SVM-based classifier for comparison.
The project was developed as part of learning machine learning and neural networks using Python.

## Files
* `network.py` - Contains the neural network implementation.
* `run_network.py` - Trains and evaluates the neural network.
* `mnist_loader.py` - Loads and preprocesses the MNIST dataset.
* `mnist_svm.py` - Implements an SVM classifier on the same dataset.
* `test_digits.py` - Used for testing digit predictions.
* `visualize_mnist.py` - Displays sample images from the dataset.
* `mnist.pkl.gz` - MNIST dataset file.

## Results
The neural network is able to classify handwritten digits from the MNIST dataset with good accuracy. The SVM implementation is included to compare performance with a traditional machine learning approach.
